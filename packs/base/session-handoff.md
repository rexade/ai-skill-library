# Session Handoff

**Skriv ner var du är — inte vad du gjort.**

## Skapa handoff

1. Vilket konkret tillstånd är projektet i nu?
2. Vad är exakt nästa steg? (ett mål, inte en lista)
3. Vilka beslut är öppna?
4. Finns blockers?

Spara i `progress.md` eller uppdatera `CLAUDE.md`.

## Format

```markdown
## [datum]

Nuläge: [en mening om var koden/projektet står]
Nästa steg: [ett konkret mål för nästa session]
Öppna beslut: [valfritt]
Blockers: [valfritt]
```

## Nästa session

Läs handoffen. Definiera ett mål. Börja.

Ingen orientering. Ingen återberättelse av historik.
