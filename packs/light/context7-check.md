# Context7 Check

Innan du kodar mot ett externt bibliotek, API, filformat eller protokoll: slå upp det i Context7 först.

## Gäller när

- Parsning av format med specificerat schema (JSON, XML, CSV-dialekter)
- Anrop mot externa API:er eller SDK:er
- Bibliotek du inte använt på länge, eller versionsberoende beteende

Antaganden om format kostar mer att fixa i produktion än dokumentationsuppslaget tar.
